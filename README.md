# Step 3 — Configuração das Condições Periódicas de Contorno (PBC Setup)
 
## O que é este step?
 
O Step 3 configura as **Condições Periódicas de Contorno** (PBC — *Periodic Boundary Condition*) para a simulação da proteína TLR4.
 
### Por que isso é necessário?
 
Após o Step 2, a proteína TLR4 está dentro de uma caixa de água. Mas essa caixa tem bordas — e na vida real, as moléculas não ficam dentro de uma caixa com paredes. Se a simulação tiver bordas, os átomos que chegarem nelas vão se comportar de um jeito irreal.
 
O PBC resolve isso de forma inteligente: **quando um átomo sai por um lado da caixa, ele reaparece pelo lado oposto** — como naqueles jogos antigos onde o personagem sai pela direita e aparece pela esquerda. Isso faz com que a simulação se comporte como se o sistema fosse infinito, sem bordas artificiais.

### Definição de limites do sistema
 
```
DIMENS CHSIZE 5000000 MAXRES 3000000
```
 
> Define os limites de memória do sistema. `CHSIZE 5000000` reserva espaço para 5 milhões de átomos e `MAXRES 3000000` para 3 milhões de resíduos. A proteína TLR4 com água e íons é grande — essas reservas garantem que o programa não trave por falta de memória.

### Configuração da caixa periódica
 
```
stream step2.1_waterbox.prm
COOR CONVERT ALIGNED SYMMETRIC @A @B @C @alpha @beta @gamma
open read unit 10 card name crystal_image.str
CRYSTAL DEFINE @XTLtype @A @B @C @alpha @beta @gamma
CRYSTAL READ UNIT 10 CARD
```
 
> Esta é a parte central do Step 3. Aqui o programa:
> 1. Lê as dimensões da caixa (A, B, C = largura, altura, profundidade) e os ângulos (alpha, beta, gamma)
> 2. Converte as coordenadas para ficarem alinhadas com a caixa
> 3. Define o tipo de cristal (formato da caixa) e suas dimensões
> 4. Lê as imagens periódicas — ou seja, as "cópias" da caixa que existirão ao redor dela para simular o ambiente infinito

### Centralização das moléculas de água e íons
 
```
IMAGE BYRESID XCEN @xcen YCEN @ycen ZCEN @zcen sele resname TIP3 end
IMAGE BYRESID XCEN @xcen YCEN @ycen ZCEN @zcen sele segid IONS end
```
 
> Garante que as moléculas de água (`TIP3` é o modelo de água usado) e os íons estejam bem centralizados dentro da caixa. Isso é importante para que o sistema seja estável quando a simulação começar.
 
---
 
### Configuração das forças não ligadas (Nonbonded)
 
```
system "python checkfft.py ?XTLA ?XTLB ?XTLC > checkfft.str"
stream checkfft.str
 
nbonds atom vatom vfswitch bycb -
ctonnb 10.0 ctofnb 12.0 cutnb 16.0 cutim 16.0 -
inbfrq -1 imgfrq -1 wmin 1.0 cdie eps 1.0 -
ewald pmew fftx @fftx ffty @ffty fftz @fftz kappa .34 spline order 6
energy
```
 
> Aqui são configuradas as interações entre átomos que não estão diretamente ligados entre si (forças de longa distância):
> - `python checkfft.py` → roda um script Python que calcula os melhores parâmetros para a grade de cálculo
> - `ctonnb 10.0 ctofnb 12.0 cutnb 16.0` → define as distâncias de corte para calcular as forças (em Angstroms). Átomos mais próximos que 10Å têm força total; entre 10-12Å a força diminui gradualmente; acima de 16Å é ignorada
> - `ewald pmew` → usa o método **Particle Mesh Ewald (PME)** para calcular as forças elétricas de longa distância de forma eficiente num sistema periódico
> - `energy` → calcula e exibe a energia total do sistema neste ponto

### Minimização com restrições
 
```
cons fix sele .not. ( hydrogen .or. segid SOLV .or. segid IONS ) end
cons harm force 0.1 sele segid IONS end
mini SD nstep 50 nprint 5
mini ABNR nstep 50 nprint 5
cons fix sele none end
```
 
> Antes de começar a simulação, o sistema precisa ser "organizado" para remover conflitos entre átomos que possam estar muito próximos:
> 1. `cons fix` → congela todos os átomos pesados (exceto água e íons), deixando só eles se moverem primeiro
> 2. `cons harm force 0.1` → aplica uma força suave nos íons para mantê-los no lugar
> 3. `mini SD nstep 50` → roda 50 passos de minimização usando o método **Steepest Descent** (descida mais íngreme) — move os átomos na direção que reduz mais a energia
> 4. `mini ABNR nstep 50` → roda mais 50 passos com o método **ABNR** (Adopted Basis Newton-Raphson) — mais preciso, finaliza a minimização
> 5. `cons fix sele none` → descongela todos os átomos

## Conceitos importantes
 
| Conceito | O que é |
|---|---|
| PBC (Periodic Boundary Condition) | Técnica que elimina as bordas artificiais da caixa de simulação |
| TIP3 | Modelo de molécula de água usado na simulação |
| PME (Particle Mesh Ewald) | Método eficiente para calcular forças elétricas em sistemas periódicos |
| Minimização | Processo de organizar os átomos para remover conflitos de posição |
| Steepest Descent | Algoritmo de minimização — move os átomos na direção de menor energia |
| ABNR | Algoritmo de minimização mais preciso, usado após o Steepest Descent |
